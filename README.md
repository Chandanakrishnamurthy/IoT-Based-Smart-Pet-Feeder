# IoT Based Smart Pet Feeder

## Description

This project is an IoT based smart pet feeder developed using ESP8266, RFID RC522, Servo Motor and Blynk IoT platform. The system helps pet owners feed their pets automatically and monitor feeding through a mobile application.

## Components Used

* ESP8266 WiFi Module
* RFID RC522
* Servo Motor
* Power Supply

## Software Used

* Arduino IDE
* Blynk IoT Platform

## Features

* RFID based pet identification
* Remote feeding using mobile app
* Servo controlled food dispensing
* Pet add and delete system
