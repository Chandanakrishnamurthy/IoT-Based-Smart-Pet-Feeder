# IoT Based Smart Pet Feeder

## Project Description

This project is an **IoT-based automatic pet feeder** developed using **ESP8266, RFID RC522, Servo Motor, and Blynk IoT platform**.
The system allows pet owners to monitor pets and control food dispensing through a mobile application.

## Components Used

* ESP8266 WiFi Module
* RFID RC522 Module
* Servo Motor
* Pet Food Container
* Power Supply

## Software Used

* Arduino IDE
* Blynk IoT Platform

## Working

* ESP8266 connects to WiFi and the Blynk mobile application.
* RFID RC522 scans the pet’s RFID tag.
* If the pet is registered, the system detects the pet.
* The servo motor can be controlled from the Blynk app to dispense food.
* The system sends notifications through Blynk events.

## Features

* RFID based pet identification
* IoT based monitoring
* Manual servo control from mobile app
* Automatic servo closing after feeding
* Pet add and delete system
