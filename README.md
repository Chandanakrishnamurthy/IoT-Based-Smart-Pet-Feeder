# IoT Based Smart Pet Feeder
## Description
This project is an IoT based smart pet feeder developed using **ESP8266**, **RFID RC522**, and **Blynk mobile application**. The system helps pet owners feed their pets automatically and remotely through the internet.
## Components Used
* ESP8266 WiFi Module
* RFID RC522
* Servo Motor
* Pet Food Container
* Power Supply
## Software Used
* Arduino IDE
* Blynk IoT App
## Working
The ESP8266 connects to WiFi and communicates with the Blynk mobile app. When the user sends a command through the app, the servo motor rotates and dispenses food for the pet. RFID is used for identification and control.
## Features
* Remote pet feeding
* Automated food dispensing
* IoT based monitoring
* Easy to use system
